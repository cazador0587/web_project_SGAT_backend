# SGAT Backend

Backend para el Sistema de Gestión de Activos Tecnológicos.

## Tecnologías

- Node.js
- Express
- MongoDB
- Mongoose
- JWT
- Bcrypt
- Joi
- Celebrate

## Instalación

npm install

## Desarrollo

npm run dev

## Producción

npm start

## Variables de entorno

PORT=
MONGODB_URI=
JWT_SECRET=
